# Delete a fixture

You can delete a fixture that shouldn't exist, for example one created by mistake or a duplicate. Deleting is for fixtures that were never meant to be there, not for matches that didn't play out as planned.

## Steps

1. Open the competition.
2. Go to the **Fixtures** section.
3. Open the fixture.
4. Choose **Delete**.
5. Confirm.

## When to delete

- A duplicate fixture.
- A fixture set up with the wrong teams that you'd rather remove than fix.
- A scheduling mistake.

## When not to delete

- **A match that's being moved.** Reschedule it instead so it keeps its history. See [Postpone or reschedule](postpone-or-reschedule.md).
- **A match that didn't play normally.** A no-show, a withdrawal, or an abandonment each has a proper outcome to record. Deleting the fixture hides what happened and can throw off the standings. See [Walkovers, withdrawals and abandonments](non-standard-outcomes.md).
- **A finalised fixture.** Its result already feeds the standings and stats. Deleting it leaves a gap. If the result is wrong, correct it rather than deleting. See [Correct a finalised result](../06-live-scoring/correct-a-finalised-result.md).

## The rule of thumb

If the fixture should never have existed, delete it. If it should exist but something about it is wrong, edit or correct it instead.
